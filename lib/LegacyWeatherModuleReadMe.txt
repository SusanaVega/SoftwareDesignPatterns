Using LegacyWeatherModule

Possible weather inputs:
"Snow"
"Rain"
"Fog"
"Normal"

Possible Aircraft Family Inputs:
"Airbus"
"Boeing"
"Bombardier"

Possible Aircraft Manuevers:
"Takeoff"
"Landing"
"Reroute"
"Circle"
"Bank"
"Tailslide"

TestWeather() returns true for a pass and false for a failed weather check. 

Status is normal upon initialization and only changes when an internal error occurs, does not change for a failed weather check, but may indicate if TestWeather is unable to be performed properly if the required properties have not been set. 

Library is provided in Java and C# with the same interface. Please contact the instructor for any issues running the library.
